SPARQL Endpoint

Query the knowledge graph at /sparql?query=...

PREFIX schema: <https://schema.org/>
SELECT ?article ?title WHERE {
  ?article a schema:Article ;
           schema:name ?title .
} LIMIT 50

About

Articles are written in Markdown under /content. A CI pipeline extracts entities and relations using an LLM, producing JSON-LD and Turtle files under /graph. This site is hosted on Azure Static Web Apps (Free plan).
